### ‚ú® Key Features

- **Configuration-Driven Content:** Most of the personal and family data is fetched dynamically from a central `config.json` file, allowing for easy updates without modifying the core HTML structure.
- **Responsive Design:** Fully optimized for all screen sizes, from mobile phones to high-resolution desktops, using modern CSS techniques.
- **Glassmorphism UI:** Features a modern aesthetic with frosted-glass headers and clean layouts.
- **SEO & Social Optimization:** includes Meta tags, Open Graph tags for rich link previews (e.g., in WhatsApp or LinkedIn), and an automatically generated `sitemap.xml`.
- **Automated Deployment:** Every push to the main branch automatically triggers a GitHub Action that builds the project and deploys it to the custom domain.
